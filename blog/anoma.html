Mô hình Mệnh lệnh (Truyền thống)

Người dùng phải chỉ định CÁCH THỨC thực hiện từng bước. Điều này phức tạp và dễ xảy ra lỗi.

// 1. Phê duyệt token USDC cho router

approve(usdc, router, 2000);

// 2. Gọi hàm hoán đổi

swapExactTokensForTokens(

  2000, // amountIn

  0.99, // amountOutMin

  [usdc, weth], // path

  userAddress, // to

  deadline // deadline

);

Hệ Sinh Thái Đối Tác Chiến Lược

Anoma tích cực hợp tác với các dự án hàng đầu để xây dựng một tương lai module và có khả năng tương tác.

A

Anoma

Lớp phối hợp phổ quát

DA

Celestia

Sử dụng restaking để cung cấp bảo mật có thể lập trình và linh hoạt.

DEX

Osmosis
